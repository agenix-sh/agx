AGENIX / AGX

AGEnts for uNIX — an agentic CLI that speaks Unix.

What is AGX?

agx is a small agent that reads from STDIN, uses a local model to plan Unix-style workflows, and executes them as pipelines of familiar tools like sort, uniq, grep, cut, tr, jq, and awk.

You describe what you want in natural language; AGX chooses a sequence of commands, executes them safely, and writes the result to STDOUT.

Install

On Linux or macOS, install or update AGX with:

curl https://agenix.sh/install.sh | sh

This script:

  • Detects your OS and architecture.
  • Downloads a prebuilt agx binary from GitHub Releases when available.
  • Falls back to building from source with cargo if needed.
  • Installs agx into a standard location (such as /usr/local/bin or $HOME/.local/bin).

You can also install from source directly with Rust:

cargo install agx

Examples

AGX behaves like a Unix filter:

cat input.txt | agx "remove duplicates" > out.txt

More examples you can try:

# Dedupe CSV rows by first three columns
cat data.csv | agx "dedupe rows based on first three CSV columns" > out.csv

# Extract id and name from a JSON array using jq
cat data.json | agx "extract id and name fields from each JSON object" > out.json
