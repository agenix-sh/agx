AGX — Agentic Unix CLI

agx is a small agent that reads from STDIN, uses a local model to plan Unix-style workflows, and executes them as pipelines of familiar tools like sort, uniq, grep, cut, tr, jq, and awk.

Install

Once DNS for agenix.sh is pointed at this site, you will be able to install AGX with:

curl https://agenix.sh/install.sh | sh

Until then, you can download binaries from GitHub Releases or build from source using cargo install.

Usage

AGX behaves like a Unix filter:

cat input.txt | agx "remove duplicates" > out.txt

For more examples, see the AGX repository:

https://github.com/agenix-sh/agx
